# Antalyayı Keşfedin

Premium landing page for antalyahakkında.com - A bridge to antalyaustası.com

## Features

- Dark, elegant, minimalist design
- Fully responsive (mobile-first)
- Smooth animations and premium effects
- Single page HTML with Tailwind CSS

## Setup

No build process required. Simply open `index.html` in a browser or deploy to any static hosting service.
